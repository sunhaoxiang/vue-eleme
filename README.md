# vue-eleme
一个使用vue.js开发的饿了么移动端商家模块WebApp，实现了点餐、评价、商品详情页、商家信息查询等功能。

## 技术栈
vue 2.0 + vue-router + vuex + axios + better-scroll

## 本地运行
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
